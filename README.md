# Authentication System Styled

This project is a styled authentication system built using HTML, Bootstrap 5, and custom CSS.  
It is an improved version of the previous HTML authentication assignment.

The application includes basic authentication pages with Bootstrap styling and responsive design.

---

## Pages Included

- Login Page
- Registration Page
- Forgot Password Page
- Reset Password Page
- Dashboard Page

---

## Technologies Used

- HTML
- Bootstrap 5
- Custom CSS

---

## Features

- Bootstrap card layout for forms
- Bootstrap form controls and buttons
- Responsive design for mobile, tablet, and desktop
- Custom CSS styling
- Navigation between pages using anchor tags
- Dashboard page with navbar and logout button
