# HTML Authentication POC

This project demonstrates a simple authentication flow using only HTML.

## Pages Included

- login.html
- register.html
- forgot-password.html
- reset-password.html
- dashboard.html

## Features

- Login redirects to Dashboard
- Register redirects to Login
- Forgot Password redirects to Reset Password
- Reset Password redirects to Login
- Logout redirects to Login

## Technology Used

- HTML only
- No CSS
- No JavaScript
